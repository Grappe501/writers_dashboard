M5.4 Build Integration — Triggering M8 Packaging From UI

Goal

Run the full pipeline (validate → index → render → package) from the cockpit.

Build Order (micro-steps)

  1. Add build button gated by readiness state.
  2. Run pipeline and stream progress logs to UI.
  3. On success, surface artifacts (zip, manifest, reports).
  4. On failure, route user to top blocking issues.

Definition of Done (DoD)

  • A build run produces a packaged reader + manifest + reports for fixtures.
